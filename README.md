El sitio web se ha creado con la finalidad de poder vender online 
los productos de la Panaderia Don fermin dandole una ventana en el comercio electronico
En el sitio web contiene una pagina de inicio, la pagina principal y la pagina de compra. 

EL SITIO WEB:

PAGINA DE INICIO: 
Esta pagina es la bienvenida del sitio web, una pagina informativa mostrando 
toda la informacion del comercio "Panaderia de Don Fermin", Esta informacion es con objetivo de 
publicitar el comercio para traer nuevos clientes, informacion relevante como una breve historia
del comercio, contactos, ubicacion y te invita a ver sus productos para comprar online.

PAGINA PRINCIPAL: 
Esta pagina es la principal, Muestra la variedad de los productos disponibles de "La Panaderia Don Fermin"
cada producto muestra una imagen del producto, unidad del producto, el precio por unidad del producto, tiene 
un boton para agregarlo al carrito y una etiqueta que muestra la cantidad del mismo producto a comprar.

PAGINA DE PAGO:
Esta pagina tiene los productos que estan en el carrito de compras y un apartado que permite pagar, muestra 
todos los datos relevantes de los productos dentro del carrito de compras, calculando el costo total. tambien 
dispone de un formulario para proceder a pagar con su respectivo metodo de pago para posterioremente enviar una 
notificación de la compra.
